# Instructions d'utilisation

## Modules disponibles

- Livres
- Usagers
- Emprunts
- Activités

Chaque module propose des actions **Ajouter / Modifier / Supprimer**.

## Livres

- **Ajouter**: bouton "Nouveau Livre"
- **Modifier**: bouton "Modifier" (ou double-clic)
- **Supprimer**: bouton "Supprimer"

## Usagers

- **Ajouter**: bouton "Nouvel Usager"
- **Modifier**: bouton "Modifier" (ou double-clic)
- **Supprimer**: bouton "Supprimer"

## Emprunts

- **Ajouter**: bouton "Nouvel Emprunt"
- **Modifier**: bouton "Modifier"
- **Supprimer**: bouton "Supprimer"

## Activités

- **Ajouter**: bouton "Nouvelle Activité"
- **Modifier**: bouton "Modifier" (ou double-clic)
- **Supprimer**: bouton "Supprimer"

## Règles de suppression

La suppression est bloquée si l’élément est lié à de l’historique :

- Livre: emprunts ou évaluations (reviews)
- Usager: emprunts, participations, évaluations, prêts d’équipement
- Activité: participations

Quand c’est bloqué, un message indique la raison.
